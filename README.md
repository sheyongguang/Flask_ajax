# 在Flask框架中使用AJAX

使用Python的Flask库时，利用AJAX方法从后台取数据。

后台数据库为Mysql，未使用sqlalchemy ,数据库连接方式不上传.

## 使用方法

请在ajax_app.py同级位置建立templates和static两个文件夹，在static文件夹下建立js文件夹。
将ajax_test.html放在templates文件夹中；
将jquery-3.1.1.min.js 放在static/js文件夹下；

## 使用环境

在windows 10 + VS code + Mysql 8 + Microsoft Edge 环境下测试通过
